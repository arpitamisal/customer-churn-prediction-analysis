# Customer Churn Prediction (Python)

This project analyzes telecom customer data to understand churn behavior and builds machine learning models to predict whether a customer will leave the service.

## Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  
- Jupyter Notebook  

## Key Work
- Exploratory data analysis of customer attributes and churn distribution  
- Visualized feature distributions such as tenure, monthly charges, and total charges  
- Identified relationships between variables using correlation heatmaps  
- Analyzed categorical features like gender, senior citizen status, and churn counts  
- Performed data preprocessing and feature preparation for modeling  
- Built and evaluated classification models to predict customer churn  

The notebook demonstrates an end-to-end machine learning workflow, from exploratory data analysis and feature understanding to predictive churn modeling.
